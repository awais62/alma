# voice-orchestration

CPU turn-taking layer for the Taible voice agent — **Pipecat / LiveKit Agents**.

Bridges the diner's browser (WebRTC) to the [`gpu-rocm`](https://github.com/Taible-io/gpu-rocm) STT→LLM→TTS pipeline: streaming glue, interruptions, turn-taking. No GPU.
